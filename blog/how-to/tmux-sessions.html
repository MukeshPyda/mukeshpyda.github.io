Tmux sessions. Run programs even when terminal closed

Install Tmux

I used Ubuntu (Debian based) to install this. 

sudo apt update
sudo apt install tmux -y

Start a New single tmux Session

If only single session needed, then: 

tmux
python3 oneway.py
Ctrl + b, then release both keys and press d (Detach)
exit and exit from the system
ssh root@ip          (connecting system again)
tmux attach          (attaches to recently detached session)

Checking tmux Sessions and connecting

Check existing list of tmux sessions 

tmux ls

When no names were given to tmux sessions. The result shows session index start from 0,1,2 etc..
To attach to one of the sessions (Session 2): 

tmux attach        (attaches to recently detached session)
tmux attach -t 2   (attaches to Session 2)

Creating and running multiple tmux Sessions

This is used when you want to keep programs organized into multiple sessions for easy access.
To create a session with name (MyFirstSession): 

tmux new -s MyFirstSession
python3 oneway.py
Ctrl + b, then release both keys and press d (Detach)
tmux attach -t MyFirstSession

You can create multiple sessions like this with appropriate names.
